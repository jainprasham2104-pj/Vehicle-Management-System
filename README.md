# Vehicle Management System

## Description
A simple Vehicle Management System developed using Python and Object-Oriented Programming (OOP) concepts. The program allows users to enter vehicle details and display information for different vehicle types such as Bike, Car, and Truck.

## Features
- Enter vehicle type
- Enter vehicle number
- Enter owner name
- Display vehicle information
- Supports Bike, Car, and Truck

## Technologies Used
- Python 3

## Concepts Used
- Classes and Objects
- Inheritance
- Constructors (__init__)
- User Input Handling
- Object-Oriented Programming (OOP)
